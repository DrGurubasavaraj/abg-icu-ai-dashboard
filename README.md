# 🫁 ICU AI Decision Support System

## Overview
An AI-powered clinical decision support tool for ICU settings, integrating ABG interpretation, ventilator management, and real-time recommendations.

## Features
- ABG Interpretation Engine (pH, compensation, mixed disorders)
- Anion Gap & Delta Ratio analysis
- ML-based decision support
- Ventilator-aware clinical override system
- ICU-style dashboard UI
- Emergency case presets

## Clinical Value
Bridges the gap between physiological reasoning and AI-assisted decision-making in critical care.

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
